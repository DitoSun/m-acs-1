# 用户反馈收集

感谢你测试 M-ACS-1。你的反馈对产品发展至关重要。请在试用后填写以下内容。

---

## 环境信息

```
操作系统版本：          （如 Ubuntu 24.04）
NVIDIA 显卡型号：       （如 RTX 4090）  
NVIDIA 驱动版本：       （nvidia-smi 输出）
内存大小：              （GB）
安装方式：              （裸机 / WSL2 / 虚拟机）
网络环境：              （中国大陆 / 海外 / 其他）
安装耗时：              （从开始到完成，分钟）
```

## 系统日志（如有故障）

```bash
# 收集安装日志
sudo cat /opt/m-acs-1/data/gpu.json

# Docker 容器状态
cd /opt/m-acs-1 && docker compose ps

# 控制面日志
cd /opt/m-acs-1 && docker compose logs control-plane --tail=30
```
